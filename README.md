# README

## サービス概要
あのキャラクターはBLで言うと受けなのか攻めなのか、  
投票によって簡単に結果がわかるアプリです。  

## 登場人物
BoysLoveが好きなすべての方  

## ユーザーが抱える問題
BoysLoveの一般論を知りたくてもなかなか周りの人に話せない  
簡単にBoysLoveの事を話したい  
よくある受けか攻めか論争に入りたい  

## 解決方法
簡単に受けか攻めか投票をして、  
該当キャラクターや芸能人が受けか攻めかBoysLove界隈の一般論が  
わかるようにする。
